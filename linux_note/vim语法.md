在**命令模式**下，以下是一些常用命令：

- _i_：进入输入模式
    
- _dd_：剪切当前行
	
- _v_:    选择字
	
- _V_：选择行
	
- _yy_：复制当前行
    
- _p_：粘贴到光标下方
    
- _u_：撤销上一次操作
    
- _:w_q ：保存退出
    
- _:q_  ：退出编辑器
    
- _:q!_：强制退出不保存修改

在**输入模式**下，可以使用_Esc_键返回命令模式。

在**底线命令模式**下，可以执行如下命令：

- _:set nu_：显示行号
    
- _:1,$s/old/new/g_：将文件中的"old"替换为"new"
	

在**命令模式**下，以下快捷键非常有用：

- _hjkl_：光标移动
    
- _Ctrl + f/b_：屏幕翻页
    
- _/word_：向下搜索"word"
    
- _?word_：向上搜索"word"
    
- _n_：重复上一次搜索
    
- _N_：反向重复上一次搜索
    
- _gg_：移动到文件第一行
    
- _G_：移动到文件最后一行
