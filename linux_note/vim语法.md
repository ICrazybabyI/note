在**输入模式**下，可以使用_Esc_键返回命令模式。

在**命令模式**下，以下是一些常用命令：

- _`i`_：进入输入模式
    
- _`dd`_：剪切当前行
	
- _`v`_:    选择字
	
- _`V`_：选择行
	
- _`yy`_：复制当前行
    
- _`p`_：粘贴到光标下方
    
- _`u`_：撤销上一次操作
    
- _`:wq`_ ：保存退出
    
- _`:q`_  ：退出编辑器
    
- _`:q!`_：强制退出不保存修改


在**底线命令模式**下，可以执行如下命令：

- _`:set nu`_：显示行号
    
- _`:set nonu`_：不显示行号
    
- _`:%s/old/new/g`_：将文件中的"old"替换为"new"
	

在**命令模式**下，以下快捷键非常有用：

- _`hjkl`_：光标移动
    
- _`Ctrl + f/b`_：屏幕翻页
    
- _`/word`_：向下搜索"word"
    
- _`?word`_：向上搜索"word"
    
- _`n`_：重复上一次搜索
    
- _`N`_：反向重复上一次搜索
    
- _`:noh`_：取消高亮搜索
    
- _`gg`_：移动到文件第一行
    
- _`G`_：移动到文件最后一行
